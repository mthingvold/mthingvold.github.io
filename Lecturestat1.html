A parameter is a numerical quantity calculated from data in a population. a statistic is a numerical quantity calculated from data in a sample.

A variable is an observable or measurable characteristi or property of indiv. objects

  • Let C be the colour of stat 245 students' eyes

types of variables Quantitative - numerical counts/measures

  • Discrete - represents that can be counted
  • Continuous - represents that which is measured Qualitative - classification based on attributes or characteristics or properties

Scales of measurement/observation

  1. Nominal - no ranked nor ordered distinctions are possible
  2. Ordinal - Observations can be ranked/ordered
  3. Interval - Observations that are also ranked or ordered. Here an equal difference in magnitude. However, a zero value does not mean nothing.
  4. Ratio - Observations are ranked or ordered, an equal difference means an equal difference in magnitude and here, a zer value means nothing.

Two Types of data collection

  1. Observational study - Data is collected straight off from a random sample. We do not influence the values of the variables in the study. Variables not controlled.
  2. Designed experiment - experiment compares treated group to a control group.

Random Sampling A simple random sample(SRS) of sizen n from population size N is such that every element in pop is equally likely of being selected to be in the sample.

  • Each element has the probability n/N of being selected
  • order of selection doesn't matter

Definition: Sampling without replacement - no element if selected may be selected more than once. w/o replacement the total number of different possible samples is

(N!)/(n!(N-n)!)
