Java provides classes for creating ggraphical applications they run on any os and that sounds disgusting

Java's JFrame class allows you to display a fram part of javax.swing

  1. Construct an object of the Jframe class ``` JFrame frame = new JFrame(); ```
  2. Set the size of the frame frame.setSize(300,400);
  3. Set the title of the frame frame.setTitle("An empty frame");
  4. Set the close operation frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
  5. Make it visible frame.setVisible(true);

Adding components You cannot draw directly on a JFrame object Instead construct an object and add it to the Frame

  • a few examples objects to add are
  • JComponent
  • JPanel
  • JTextComponent
  • JLabel If you have more than one component, put them into a panel First create components JButton button = new JButton("Click me!"); JLabel label = new JLabel("Hello, World!"); JPanel panel = new JPanel(); //then add them to the panel panel.add(button); panel.add(label); frame.add(panel);

Using Inheritance to customize Frames

  • For complex frames:
    • Design a subclass of JFrame
    • Store the components as instance variables
    • initialize them in the constructor of your subclass.
